DISTRIBUTION NOTES
------------------
- This bundle intentionally removes `pillow-heif`, which was the clearest
  copyleft blocker in the earlier build.
- macOS HEIC / HEIF handling in this bundle is done through the built in `sips`
  command rather than a bundled Python HEIF library.
- If you ship binary builds that bundle `opencv-python` or `pypdfium2`, preserve
  the wheel-specific third-party notice files from the exact wheels you used.
- If you distribute source code only, keep this file and the full `licenses/`
  directory in the repository.

Abaku_OpenSourceLic macOS wrapper

This is an unsigned macOS .app wrapper around the commercial-safer Python source bundle.

Note:
- This app is NOT Apple-signed or notarized. I cannot produce an Apple-signed app without your Apple Developer certificate and provisioning setup.
- First launch may require right-click > Open.
- The wrapper copies the embedded source to ~/stego-lab/abaku_opensourcelic_app and runs the setup script there.
